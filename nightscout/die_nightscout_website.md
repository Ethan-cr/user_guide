# Die Nightscout Website

Die Nightscout Website ist die zentrale Website zur Darstellung und Speicherung von Diabetes - Daten. Sie kann als komplettes Tagebuch mit Eingabe von Insulin, Kohlenhydraten, Bewegung uvm. verwendet werden. Der geasamte CGM - Verlauf wird online dargestellt und zyklisch aktualisiert. Es gibt über Plugins umfangreiche Konfigurationsmöglichkeiten.
Das Reporting bietet die Möglichkeit, Daten für andere zur Analyse auszugeben.
